Proyecto MDIEstudiantes
.NET Framework: 4.7.2
Abre la solución MDIEstudiantes.sln con Visual Studio 2022 y compila.
Descripción:
- Form1: MDI Padre con ToolStrip (Ingresar Estudiante, Ver Estudiantes, Ver Mejores Promedios)
- Form2: Ingreso de estudiantes y sus asignaturas
- Form3: Listado de estudiantes con promedio (doble clic para ver detalles)
- Form4: Gráfico de promedios (Chart)
Notas:
- El proyecto guarda los estudiantes en memoria (DatosCompartidos.Estudiantes).
- Listo para compilar y ejecutar.
